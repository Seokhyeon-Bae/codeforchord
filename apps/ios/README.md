# CodeForChord — Native iOS (SwiftUI)

Open in Xcode:

```bash
open CodeForChord.xcodeproj
```

Run the FastAPI backend on port **8000**, then build and run on a simulator. Configure the API URL under the in-app **Settings** (gear) if needed.

Full notes: [docs/XCODE_AND_REMOTE.md](docs/XCODE_AND_REMOTE.md).
